# Fan-Game-Arknights
A web-based game database interface for browsing and filtering character data in this fan-made game.

## 功能
- **搜索** – 支持全局关键词搜索（选手名、等级、伤害类型、抗性、天赋等任意字段）
- **筛选** – 多种筛选选项，用于缩小游戏选择范围
- **排序** – 按不同标准对各角色进行排序
- **数据表** – 清晰简洁的游戏数据表格展示
- **响应式设计** – 界面友好，操作直观

## 使用方法
使用搜索栏查找特定游戏
通过筛选工具栏应用筛选条件
点击列标题对结果进行排序
在数据表中查看详细信息

## 技术细节
- 单页 HTML 应用程序
- 基于 JavaScript 的筛选和排序功能
- 采用 CSS 样式设计，呈现现代外观
